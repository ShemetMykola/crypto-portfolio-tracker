## Example Portfolio
```json
{
  "bitcoin": 1.5,
  "ethereum": 10,
  "solana": 50
}
```

## How to Use
1. Install dependencies:
```bash
pip install requests pandas matplotlib
```
2. Update `portfolio.json` with your holdings.
3. Run `portfolio_tracker.py` to see portfolio value and graphs.
